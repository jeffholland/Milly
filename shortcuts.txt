All keyboard shortcuts usable in the Text widget:


return                 submit entry
shift+return           new line of text
cmd+i                  insert entry
cmd+s                  save
cmd+l                  load
cmd+c                  clear
cmd+backspace          delete top entry
cmd+shift+backspace    delete bottom entry
cmd+e                  enter edit selection mode


EDIT SELECTION MODE

Use up and down arrow keys to navigate the edit buttons of all entries.
Press return to select one.
